## ✨ Features

| Feature | Description |
|---|---|
| 🎵 **Local Library** | Play MP3 files from your local `/musica` folder |
| ▶️ **YouTube Streaming** | Search and stream any YouTube video — no ads, no downloads needed |
| 🟠 **SoundCloud** | Search and stream from SoundCloud (API key required) |
| 📋 **Playlists** | Create custom playlists mixing local and YouTube tracks |
| 📚 **YouTube Library** | Save YouTube tracks to your library without downloading |
| 🎨 **6 Color Themes** | Crimson, Ocean, Emerald, Violet, Amber, Light |
| 🌈 **Dynamic Color** | UI tints based on the current album cover |
| 🎛️ **10-Band EQ** | Full equalizer with per-band control |
| 📊 **Visualizer** | Bars, wave, and circle spectrum modes |
| 🖥️ **Fullscreen Visualizer** | Immersive fullscreen audio visualization |
| 💿 **Vinyl Spin** | Album art rotates while playing |
| 🔀 **Crossfade** | Smooth transitions between tracks |
| 😴 **Sleep Timer** | Auto-stop after 15, 30, 45, 60 or 90 minutes |
| ⚡ **Speed Control** | Playback speed from 0.5x to 2x |
| 🔖 **Bookmarks** | Mark exact timestamps in any song |
| 📋 **Queue** | Build and reorder a playback queue |
| 🎤 **Lyrics** | Sync `.lrc` files with your music |
| 📈 **Stats** | Track play counts and listening time |
| 🎹 **Media Keys** | Global keyboard shortcuts work even when minimized |
| 🪟 **Mini Player** | Always-on-top compact player window |
| ⬇️ **Download from YT** | Save any YouTube track as MP3 to your library |

## 📦 Installation

### Requirements

Before installing RatPlayer, make sure you have:

- **Python 3.x** — [Download here](https://www.python.org/downloads/)
- **yt-dlp** — For YouTube streaming

```bash
# Install yt-dlp after installing Python
python -m pip install yt-dlp
```

### Install RatPlayer

1. Download the latest **RatPlayer Setup.exe** from [Releases](https://github.com/Gerrrrp1901/RatPlayer/releases/latest)
2. Run the installer
3. Launch RatPlayer from your desktop shortcut

---

## 🎵 Adding Music

**Local files:** Drop your `.mp3` files into the `musica` folder inside the installation directory.

**YouTube:** Use the YouTube tab in the sidebar to search and stream directly — or save tracks to your library without downloading.

**Drag & Drop:** Drag `.mp3` files directly into the app window.

## ⌨️ Keyboard Shortcuts

| Key | Action |
|---|---|
| `Space` | Play / Pause |
| `→` | Next track |
| `←` | Previous track |
| `↑` / `↓` | Volume up / down |
| `M` | Mute |
| `F` | Toggle favorite |
| `B` | Bookmark current position |
| `Q` | Open queue |
| `Esc` | Close fullscreen visualizer |
